# Logistik - Management Pro

Plataforma integral para la gestión de renta de mobiliario, control de inventarios, clientes y planificación de eventos asistida por IA.

## Características Principales
- **PWA (Progressive Web App)**: Aplicación nativa instalable con soporte offline completo.
- **Gestión Multi-usuario**: Sistema de roles para gerencia y personal operativo.
- **Control Logístico**: Guías de remisión, control de despacho y gestión de devoluciones con actas de faltantes.
- **Contabilidad y Finanzas**: Libro diario, registro de abonos, facturación y gestión de retenciones.
- **Actualización Automática**: Sistema conectado a GitHub para despliegue continuo.

## Despliegue en GitHub
Este repositorio está configurado con **GitHub Actions** para desplegar automáticamente cualquier cambio en la rama `main` hacia GitHub Pages o servidores estáticos.

### Cómo actualizar
Cualquier cambio guardado y enviado (push) al repositorio activará el Service Worker v12 para refrescar la aplicación en todos los dispositivos instalados.
